# EducationalAdministrationSystem
 Java毕业设计-基于SSM框架的教务管理系统
<h2>一、项目简介</h2>

<h2>二、功能介绍</h2>
<div class="markdown-heading" dir="auto">
<div class="markdown-heading" dir="auto">
<h3 class="heading-element" dir="auto" tabindex="-1">小程序</h3>
</div>
<ul dir="auto">
 	<li>基本操作：登录、修改密码、获取个人信息、修改个人信息
所有课程：获取课程列表、选课、筛选课程、获取课程详情
已选课程：获取已选课程列表、获取课程信息、退课
已修课程：获取已修课程列表、获取课程信息、获取课程成绩</li>
</ul>
<div class="markdown-heading" dir="auto">
<h3 class="heading-element" dir="auto" tabindex="-1">管理后台</h3>
</div>
<ul dir="auto">
 	<li>基本操作：登录、修改密码、修改密码、重置其他账号密码
课程管理：筛选课程、添加课程信息、修改课程信息、删除课程、获取课程列表、查看课程详情
学生管理：筛选学生、添加学生信息、获取学生信息列表、查看学生信息详情、删除学生、修改学生信息
教师管理：筛选教师、添加教师信息、获取教师信息列表、查看教师信息详情、删除教师、修改教师信息</li>
</ul>
</div>
